# secKillBot

一个基于 Java 的秒杀机器人。

## 技术栈

- Java 8
- Maven 3.6.3
- OkHttp（HTTP 请求）
- Gson（JSON 解析）
- Logback（日志）
- JUnit 5（测试）

## 构建与运行

```bash
mvn clean package          # 打包
mvn test                   # 运行测试
java -jar target/secKillBot-1.0-SNAPSHOT-jar-with-dependencies.jar  # 运行
```
